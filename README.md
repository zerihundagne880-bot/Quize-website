# 🧠 Interactive Quiz App
ይህ በ HTML5, CSS3 እና JavaScript የተሰራ ዘመናዊ የጥያቄና መልስ ድረ-ገጽ ነው። ተጠቃሚዎች እውቀታቸውን እንዲፈትኑ እና ውጤታቸውን ወዲያውኑ እንዲያዩ ይረዳል።

### ✨ ዋና ዋና ተግባራት (Key Features)
- ⏱ **Dynamic Questions:** ጥያቄዎቹን በጃቫስክሪፕት በየተራ ያመጣል።
- ✅ **Instant Feedback:** መልሱ ልክ ይሁን አይሁን ወዲያውኑ በቀለም (አረንጓዴ/ቀይ) ያሳያል።
- 📊 **Score Board:** መጨረሻ ላይ ተጠቃሚው ስንት እንደመለሰ ውጤት ያሳያል።
- 🔄 **Restart Quiz:** ውጤት ከታየ በኋላ ድጋሚ መጀመር የሚያስችል ቁልፍ።

---

### 📂 የፋይሎች ዝርዝር
- `index.html` - የጥያቄዎቹ ሳጥን እና የውጤት ገጽ።
- `style.css` - ዘመናዊ እና ግልጽ የሆነ የዲዛይን ገጽታ።
- `script.js` - ጥያቄዎቹን የሚቆጣጠር እና ውጤት የሚያሰላ Logic።
