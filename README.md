### ✨ Key Features

*   ⏱ **Dynamic Question Loading:** Questions are injected into the DOM dynamically via JavaScript arrays.
*   ✅ **Instant Feedback:** Visual cues (Green for correct, Red for incorrect) provide immediate results for each choice.
*   📊 **Real-time Scoring:** The system tracks user progress and displays a final performance summary.
*   🔄 **Restart Functionality:** Allows users to reset the session and try the quiz again instantly.
*   📱 **Responsive Design:** Works perfectly on mobile devices, tablets, and desktops.

---

### 📂 Project Structure

*   `index.html` — Contains the semantic container for questions and the result dashboard.
*   `style.css` — Features a clean, modern design with a focus on readability and active button states.
*   `script.js` — Handles the core quiz logic, array-based question storage, and score calculation.
