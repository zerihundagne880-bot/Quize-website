### ✨ Key Features
- ⏱ **Dynamic Question Loading:** Questions are injected into the DOM dynamically via JavaScript.
- ✅ **Instant Feedback:** Visual cues (Green for correct, Red for incorrect) provide immediate results for each answer.
- 📊 **Real-time Scoring:** Tracks user progress and displays a final performance summary.
- 🔄 **Restart Functionality:** Allows users to reset the session and try the quiz again instantly.

---

### 📂 Project Structure
- `index.html` — Contains the semantic container for questions and the result dashboard.
- `style.css` — Features a clean, modern design with a focus on readability and button states.
- `script.js` — Handles the quiz logic, array-based question storage, and score calculation.
